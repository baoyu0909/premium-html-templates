# Premium HTML Templates Skill 使用说明

这个技能包用于生成高贴合度的 HTML 汇报页、PPT 风格 deck、单页 dashboard 和信息卡页面。使用时重点说清楚三件事：输出模式、风格类型、色彩倾向。

## 推荐调用格式

```text
请使用 Premium HTML Templates Skill 生成一个 HTML 页面。

输出模式：deck / single-page
风格类型：light-report / dark-ip / light-glass
色彩倾向：red-primary / blue-primary / teal-primary / mixed-accent
主题：这里写你的主题
内容：这里写你的素材、数据、章节或要求
输出要求：生成一个可直接打开的完整 HTML 文件
```

## 输出模式怎么选

### deck

适合 PPT、分页汇报、复盘、提案、方案、路演。

调用时可以这样说：

```text
输出模式：deck
要求：使用 16:9 分页结构，每页内容必须完整放在画布内，不要使用内部滚动、缓冲区或裁切。
```

### single-page

适合单页 dashboard、信息卡页面、资源盘点、渠道对比、长页面。

调用时可以这样说：

```text
输出模式：single-page
要求：页面和卡片高度可以随内容自然增长，不要强制 16:9，保持清晰的 section、网格和卡片节奏。
```

## 风格类型怎么选

### light-report

白底商务汇报风格，适合正式汇报、经营复盘、项目提案、SOP、排期规划、数据总结。

可选色彩倾向：

- `red-primary`：红色主视觉，适合强调增长、结论、重点动作、正式决策。
- `blue-primary`：蓝色主视觉，适合数据、系统、科技、稳定感表达。
- `mixed-accent`：红蓝绿混合强调，适合综合型管理汇报。

### dark-ip

深色科技 IP 风格，适合个人 IP、品牌策略、视觉体系、形象升级、科技感方案。

可选色彩倾向：

- `blue-primary`：冷蓝科技感。
- `teal-primary`：青绿色高级感。
- `mixed-accent`：蓝紫青多色光感。

### light-glass

浅色玻璃拟态信息卡风格，适合资源盘点、渠道对比、商务沟通、报价解读、招商信息。

可选色彩倾向：

- `blue-primary`：清爽商务信息卡。
- `teal-primary`：柔和清透信息卡。
- `mixed-accent`：蓝、粉、橙柔光组合。

## 示例 1：红色主视觉商务汇报

```text
请使用 Premium HTML Templates Skill 生成一个 HTML 汇报页。

输出模式：deck
风格类型：light-report
色彩倾向：red-primary
主题：视频号数据复盘与增长建议

内容：
1. 本月视频号整体数据表现
2. 内容播放、互动、私信、线索转化变化
3. 爆款内容结构拆解
4. 下阶段增长动作和执行排期

要求：
1. 使用 16:9 分页 deck 结构
2. 红色作为主强调色，蓝色只作为辅助信息色
3. 每页标题和上方保留安全间距
4. 每页内容必须完整落在画布内
5. 不要使用内部滚动、缓冲区或裁切
6. 不要添加测试页标题、风格说明栏或 demo 导航
7. 输出一个可直接打开的完整 HTML 文件
```

## 必须避免的输出

调用时可以加上这段，减少生成偏差：

```text
不要添加顶部测试说明栏、风格演示导航按钮、“这是第几种风格”的外层说明区，也不要添加和正式内容无关的 demo 壳。页面应直接进入正式内容。
```

如果是 `deck`，建议额外加上：

```text
每页必须完整落在 16:9 画布内。内容太多时请主动拆页，不要缩到看不清，不要裁切，不要给单页内部加滚动条。
```

如果是 `single-page`，建议额外加上：

```text
不要强制 16:9。页面和卡片可以自然变长，但要拆成清晰的 section 和卡片，不要把一个卡片写成大段文章。
```
