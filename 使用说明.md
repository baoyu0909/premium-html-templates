# Premium HTML Templates Skill 使用说明

这个技能包的目标很简单：用户不用懂参数，也不用指定复杂指令，只要说清楚“想做什么内容”，系统会自动推荐最合适的页面形式、风格和配色，并直接生成 HTML。

## 最简单的用法

直接这样说就可以：

```text
帮我生成一个关于视频号数据复盘的 HTML 汇报页。
```

或者：

```text
帮我把下面这些内容做成一个好看的 HTML 页面：

主题：渠道资源盘点
内容：
1. 渠道分类
2. 各渠道资源价值
3. 合作成本
4. 转化潜力
5. 推荐优先级
```

系统会自动判断：

- 是做成 PPT 式分页汇报，还是单页信息页面
- 用白底商务、深色科技，还是浅色玻璃卡片风格
- 主色用红色、蓝色、青绿色，还是混合强调色
- 内容太多时是拆页，还是拆成多个卡片模块

## 推荐你这样写

### 只给一个主题

```text
帮我生成一个关于视频号数据汇报的 HTML 文件。
```

适合你还没有整理详细内容，只想先让系统自动生成一版完整方案。

### 给主题和大纲

```text
帮我生成一个关于视频号数据复盘的 HTML 汇报页。

大纲：
1. 本月核心数据
2. 爆款内容拆解
3. 用户互动和私信线索
4. 当前问题
5. 下阶段增长建议
```

适合你已经知道大概结构，希望系统自动补足表达、布局和视觉。

### 给完整内容

```text
帮我把下面内容做成一个 HTML 汇报页，风格高级一点，适合对外展示。

内容：
这里粘贴你的完整文字、数据、表格、结论或方案。
```

适合你已经有素材，希望系统帮你重新组织成更好看的 HTML。

## 常见场景怎么说

### 正式汇报

```text
帮我生成一个关于季度经营复盘的 HTML 汇报页，适合正式汇报。
```

系统通常会自动选择：白底商务风格、红色或蓝色重点强调、PPT 式分页结构。

### 数据复盘

```text
帮我生成一个关于账号运营数据复盘的 HTML 汇报页，重点突出数据变化、问题和下一步动作。
```

系统通常会自动选择：白底商务风格、数据卡片、趋势拆解、行动建议模块。

### 增长方案

```text
帮我生成一个关于视频号增长方案的 HTML 汇报页，重点突出目标、方法、执行排期和预期结果。
```

系统通常会自动选择：红色主视觉、强结论标题、阶段拆解、执行路径。

### 个人 IP 或品牌方案

```text
帮我生成一个老板个人 IP 视觉升级方案 HTML 页面，要有科技感和高级感。
```

系统通常会自动选择：深色科技风格、强标题、冷色高光、品牌策略结构。

### 资源盘点或渠道对比

```text
帮我生成一个渠道资源盘点 HTML 页面，内容包括资源价值、合作成本、转化潜力和推荐优先级。
```

系统通常会自动选择：浅色玻璃卡片风格、单页信息结构、卡片分组展示。

### 单页数据看板

```text
帮我生成一个账号运营数据看板 HTML 页面，包含播放、互动、私信、线索和成交数据。
```

系统通常会自动选择：单页 dashboard、商务卡片、指标总览、图表区域和行动建议。

## 如果你想明确控制风格

大多数时候不用写这些。但如果你有明确偏好，可以直接用中文说：

```text
做成白底商务风格，红色作为主强调色。
```

```text
做成深色科技风格，更适合个人 IP 方案。
```

```text
做成浅色玻璃卡片风格，不要强制 16:9，适合单页展示。
```

```text
做成 PPT 式分页汇报，每一页都要完整显示，不要出现内部滚动。
```

## 自动判断规则

当你没有指定风格时，系统会按下面的方式自动选择：

| 你输入的内容 | 系统优先选择 |
| --- | --- |
| 汇报、复盘、提案、规划、方案 | PPT 式分页汇报 |
| 单页、看板、dashboard、资源盘点、渠道对比 | 单页信息页面 |
| 经营、数据、SOP、排期、增长建议 | 白底商务风格 |
| IP、品牌、视觉升级、科技感、高级感 | 深色科技风格 |
| 资源、渠道、报价、对比、招商信息 | 浅色玻璃卡片风格 |
| 增长、重点动作、关键结论 | 红色主强调 |
| 数据、系统、稳定、分析 | 蓝色主强调 |
| 信息卡、资源盘点、多模块展示 | 混合强调色 |

## 生成时默认会避免

系统默认不会添加这些无关内容：

- 顶部测试说明栏
- 风格演示导航按钮
- “这是第几种风格”的说明区
- 和正式内容无关的 demo 外壳
- 暴露内部来源或历史文件名

## 最推荐的一句话

如果你不知道怎么写，直接用这句：

```text
帮我根据下面内容，自动选择最合适的风格、配色和布局，生成一个高级、完整、可直接打开的 HTML 页面。

内容：
这里粘贴你的主题、素材或大纲。
```
