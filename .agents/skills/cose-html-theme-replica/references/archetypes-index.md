# Archetypes Index

这个 skill 当前收敛为三种风格原型，均由内部历史案例提炼而来。

## `light-report`

- 内部来源：历史商务汇报与方案型案例
- 关键词：
  - 白底商务
  - 正式汇报
  - 16:9 滚动式 PPT
  - 红橙蓝点缀
  - 分章节强结构
- 典型场景：
  - 汇报方案
  - 项目排期
  - SOP
  - 老板汇报
  - 策略提案

读取文件： [variant-light-report.md](variant-light-report.md)

## `dark-ip`

- 内部来源：历史品牌/IP 策略型案例
- 关键词：
  - 深色科技
  - 玻璃层叠
  - 品牌/IP方案
  - 冷色霓虹点缀
  - 高级感
- 典型场景：
  - 个人 IP
  - 科技品牌
  - 视觉方向
  - 创意提案
  - 风格矩阵

读取文件： [variant-dark-ip.md](variant-dark-ip.md)

## `light-glass`

- 内部来源：历史商务沟通与资源盘点型案例
- 关键词：
  - 浅色玻璃拟态
  - 信息卡
  - 双栏对比
  - 柔和彩色弥散背景
  - 沟通汇总
- 典型场景：
  - 渠道对比
  - 展会资源
  - 商务方案
  - 报价拆解
  - 信息摘要

读取文件： [variant-light-glass.md](variant-light-glass.md)

## 选择建议

- 追求“像老板汇报 PPT”时选 `light-report`
- 追求“像科技品牌策略 deck”时选 `dark-ip`
- 追求“像信息卡+对比展示页”时选 `light-glass`

## 与 mode 的组合

- `light-report + deck`：正式汇报、经营复盘、项目提案。
- `dark-ip + deck`：品牌策略、个人 IP、视觉策略。
- `light-glass + single-page`：资源盘点、渠道对比、商务信息卡。
- `light-report + single-page`：商务 dashboard、单页数据简报。

## 使用提醒

- 这里保留的是风格抽象，不是对外可见的来源说明。
- 即使内部样板文件被删除，这三种原型定义也应视为稳定可用。
